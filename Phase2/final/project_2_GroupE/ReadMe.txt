Before running our code, please make sure that you have installed package Orange3-Associate, Orange3 and pandas_ml. If you have not installed them, you can install them as follow steps.
1.download pip
2.open terminal 
3.pip install Orange3-Associate
  pip install Orange3
  pip install pandas_ml

We put the code of association rules in the folder of Assocaition Rules. predictive_analysis.py is the code of all predictive analysis of project 2.
